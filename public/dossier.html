HOW TO USE THIS
For Designers (Michelle, team)
Relay Chat — Go to ryujin.inferis.app and talk to the agents. Describe what you need in plain English. Chief will route it.
Arg0n.dev — Open arg0n.dev to generate videos/images/audio directly. Pick a model, type a prompt, hit generate. Check the Guide tab for model recommendations.
Briefs — Upload a creative brief via the relay. Chief decomposes it into shots, frames, audio, and design assets. Each gets assigned to the right agent and model.
Keyframes — Upload a start/end frame on arg0n.dev to anchor video generation. Your product shots become the starting point, not an AI guess.
For Developers
API — All endpoints at ryujin.inferis.app/api/argon/*. See ARGON-DEV-GUIDE.md in the repo.
Repo — github.com/ChopperD00/deadweight-argon (arg0n.dev UI) + github.com/ChopperD00/inferis (core engine).
Claude Code — SSH to ACIDBURN, cd ~/arg0n-dev, run claude. Reads CLAUDE.md + COWORK-BRIEF.md for context.
Deploy — Push to main → Vercel auto-deploys to arg0n.dev in ~30s.
For Phil (Renzo access)
This chat — Talk to Renzo here. Full mesh access, deployment, API wiring, file creation, everything.
ForgeChief — ryujin.inferis.app relay. For client-facing creative orchestration.
Arg0n.dev — For quick generation runs and demos.
Cowork/Code — For autonomous parallel builds on ACIDBURN.
